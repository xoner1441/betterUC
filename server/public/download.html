Auto-Updater

Ingame aktualisieren

Wenn betterUC eine neue Version erkennt, kannst du das Update direkt im Chat vorbereiten. Die neue Jar wird heruntergeladen und ersetzt die alte Datei, sobald Minecraft geschlossen wurde.

Der Auto-Updater verwendet ebenfalls die betterUC-Downloadstrecke.
